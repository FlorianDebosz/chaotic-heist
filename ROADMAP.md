PHASE 1: Fonctionnel (5 semaines)

SEMAINE 1: Menu + Lobby
  - Menu principal (Play, Settings, Quit)
  - Créer lobby
  - Rejoindre lobby (code ou amis)
  - Voir les joueurs dans le lobby
  
SEMAINE 2: Perso + Réseau
  - Choisir personnage (voir ce que les autres prennent)
  - Contrôles customisables (rebind keys)
  - Level charge pour tout le monde ensemble
  - Les joueurs se voient
  
SEMAINE 3: Le Joueur Bouge
  - Joueur peut bouger (haut/bas/gauche/droite)
  - Sauter
  - Ramasser objet (diamant)
  - Les autres joueurs voient tes actions
  
SEMAINE 4: Musée + Gardes
  - Musée simple (2-3 salles, juste basic layout)
  - Gardes patrol
  - Gardes te voient = te chassent
  - Timer (host peut l'activer, durée custom)
  
SEMAINE 5: Finish Base + Test
  - Complétement jouable (spawn → vol → escape)
  - Tester à 4 joueurs réels
  - Fix bugs évidents
  - Version prêt à montrer à des potes
